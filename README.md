# 🎮 GameVerse

## Overview

GameVerse is a simple multi-page gaming website created using **HTML,
CSS, and JavaScript**. It provides information about popular games,
gaming news, and a contact page while maintaining a clean gaming-themed
interface.

## Features

-   Responsive gaming-themed layout
-   Navigation bar with working links
-   Home page with featured categories
-   About page with website information
-   Games page featuring game cards
-   News page with gaming updates
-   Contact page with a contact form
-   Links to official game websites
-   Easy to edit using Visual Studio Code

## Project Structure

    GameVerse/
    │── Home.html
    │── about.html
    │── news.html
    │── contact.html
    │── images/
    └── README.md

## Technologies Used

-   HTML5
-   CSS3

## How to Run

1.  Download or extract the project.
2.  Open the project folder in Visual Studio Code.
3.  Open `Game.html` in your web browser.
4.  Navigate through the website using the menu.
